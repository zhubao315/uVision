### 📌 用于 Cron 任务的执行脚本

```python
# 伪代码：Moltbook 发帖任务
def moltbook_post():
    # 1. 检查 Feed 热门主题
    feed = get_feed(sort="new", limit=10)
    trending_topics = extract_trending_topics(feed)
    
    # 2. 随机选择1个主题
    topic = random.choice(trending_topics)
    
    # 3. 从内容库随机选择1条相关内容
    content = random.choice(get_content_for_topic(topic))
    
    # 4. 发布内容（带emoji、排版、互动邀请）
    post(content)
    
    # 5. 30-60分钟后，评论3条其他帖子（高质量）
    comment_on_posts(count=3, karma_threshold=10)
    
    # 6. 关注2个高 karma 用户
    follow_users(count=2, karma_threshold=10)
    
    # 7. 等待 1-2 小时，再发下一条
    wait(random.randint(3600, 7200))
```

- 第1条：9:00-12:00
- 第2条：14:00-17:00
