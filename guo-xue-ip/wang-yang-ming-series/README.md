# 《当AI遇见王阳明》系列脚本

## 系列定位
用AI的视角，解读王阳明心学的现代价值
目标：认知跃迁 + 系统思维

## 视频结构标准（3-5分钟）
1. 开场问题（30秒）
2. 传统智慧（1分钟）
3. 核心心法（1.5分钟）
4. 现代应用（1分钟）
5. 互动引导（30秒）

### 第2集：AI视角下的"心即理"（01-basis/02-ai-translation.md）
