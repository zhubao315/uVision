# 《王阳明心学：AI时代的认知升级课》
## 配套物料清单（符合得到APP标准）

### 一、核心交付物

#### 1. 课程体系（12讲）
- 第1讲：王阳明是谁？——AI时代的哲学家
- 第2讲：心即理——你的认知框架决定现实
- 第3讲：知行合一——从知道到做到的行动密码
- 第4讲：致良知——AI时代的内在指南针
- 第5讲：心学AI框架整合——从混沌到秩序
- 第6-12讲：待补充（中庸、易经等拓展内容）

#### 2. 逐字稿（每讲20-30分钟）
- 每讲包含5部分：
  - 【开场】（30秒）：问题导入
  - 【第一部分】（3分钟）：理论讲解
  - 【第二部分】（3分钟）：案例分析
  - 【第三部分】（2分钟）：行动指南
  - 【结尾】（1分钟）：总结+预告

#### 3. 配套物料清单

| 物料类型 | 说明 | 交付形式 |
|---------|------|---------|
| 课程大纲 | 12讲总览 + 每讲要点 | PPT/PDF |
| 学员手册 | 每讲重点笔记 + 思考题 | PDF |
| 知识图谱 | 心学AI框架的脑图 | PNG/PDF |
| 案例库 | 职场应用案例集 | Excel/PDF |
| 课后作业 | 每讲1个实践任务 | PDF |

### 二、课程特点

#### 1. 结构化
- "问题导入 → 理论讲解 → 案例分析 → 行动指南"四步法
- 每讲独立成章，又环环相扣

#### 2. 实战性
- 每讲包含"职场应用案例"
- 每讲提供"实践任务"

#### 3. 可视化
- 知识图谱：心学AI框架的脑图
- 对比表格：心学 vs AI的映射关系
- 流程图：知行合一的闭环模型
