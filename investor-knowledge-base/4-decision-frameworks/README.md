## 📊 风险评估框架

### 风险类型
**核心维度**:
1. **技术风险**: 技术迭代、替代品、研发失败
2. **市场风险**: 需求萎缩、竞争加剧、价格下跌
3. **运营风险**: 管理层、执行力、供应链
4. **财务风险**: 现金流、杠杆、偿债能力
5. **政策风险**: 监管、税收、贸易、地缘政治

### 风险矩阵
| 风险等级 | 概率高 | 概率中 | 概率低 |
|--|--|--|--|
| **影响高** | 红色（立即行动） | 橙色（重点关注） | 黄色（监控） |
| **影响中** | 橙色（重点关注） | 黄色（监控） | 绿色（常规） |
| **影响低** | 黄色（监控） | 绿色（常规） | 绿色（常规） |

### 风险缓解措施
**核心策略**:
1. **分散化**: 行业、地区、资产类别
2. **对冲**: 金融工具、保险、多元化
3. **止损**: 单笔亏损 ≤ 5%，总亏损 ≤ 20%
4. **压力测试**: 极端情景下的损益预测

## 📊 估值模型框架

### 现金流折现 (DCF)
**核心公式**:
```
价值 = Σ (FCF_t / (1 + WACC)^t) + TV / (1 + WACC)^n
```

**关键参数**:
- **FCF (自由现金流)**: 经营现金流 - 资本支出
- **WACC (加权资本成本)**: 权益成本 × 权重 + 债务成本 × (1 - 税率) × 权重
- **永续增长率 (g)**: 通常取GDP增长率 (2-3%)
- **终端价值 (TV)**: FCF_n × (1 + g) / (WACC - g)

**适用场景**: 成熟企业、稳定现金流、低增长行业

**风险**: 参数敏感性、预测偏差、WACC估算

---

### 相对估值
**核心指标**:
- **PE (市盈率)**: 市值 / 净利润
- **PB (市净率)**: 市值 / 净资产
- **PS (市销率)**: 市值 / 收入
- **EV/EBITDA**: (市值 + 净负债) / 息税折旧摊销前利润

**适用场景**: 成长企业、亏损企业、可比公司多

**风险**: 估值倍数选择、可比性、周期性影响

---

### 期权定价
**核心模型**: Black-Scholes、二叉树、蒙特卡洛

**适用场景**: 早期企业、期权估值、实物期权

**核心参数**:
- **标的资产价值**: 公司当前价值
- **行权价格**: 投资成本
- **到期时间**: 投资期限
- **波动率**: 价值波动程度
- **无风险利率**: 国债收益率

## 📊 投后管理框架

### KPI监控
**核心指标**:
- **财务KPI**: 收入、利润、现金流、ROE
- **运营KPI**: 用户数、市场份额、客户满意度
- **战略KPI**: 产品进展、合作伙伴、市场份额

**监控频率**:
- 日度: 现金流、用户数
- 周度: 销售线索、转化率
- 月度: 财务报表、运营指标
- 季度: 战略回顾、调整计划

---

### 战略支持
**核心价值**:
1. **资源对接**: 客户、供应商、合作伙伴
2. **人才引入**: 管理层、核心技术、销售团队
3. **战略建议**: 市场定位、产品路线、商业模式
4. **退出规划**: 上市、并购、回购、清算

---

### 退出规划
**核心策略**:
1. **IPO**: 市场环境、估值水平、投资者兴趣
2. **并购**: 战略买家、财务买家、交易结构
3. **回购**: 现金流、估值水平、股东意愿
4. **清算**: 破产、资产出售、债务重组

**退出时机**:
- 估值高点：PE历史分位数 > 75%
- 成长放缓：收入增速 < 10%
- 战略契合：并购方出现、上市窗口

## 🔄 决策流程框架

### Step 1: 决策启动
- 机会识别：市场误定价、行业颠覆、事件驱动
- 初步评估：市场规模、竞争格局、团队能力
- 决策委员会：投资委员会、风险控制委员会

### Step 2: 尽职调查
- 财务尽调：审计报告、财务分析
- 法律尽调：合同、诉讼、知识产权
- 商业尽调：市场、客户、竞争对手
- 管理层尽调：访谈、履历、历史业绩

### Step 3: 投资决策
- 估值分析：DCF、相对估值、期权定价
- 风险评估：风险矩阵、压力测试、敏感性分析
- 交易结构：股权比例、估值、对赌、退出

### Step 4: 投资执行
- 合同签署：投资协议、股东协议、公司章程
- 资金划付：分期付款、交割条件
- 资源对接：客户、人才、战略支持

### Step 5: 持续跟踪
- KPI监控：财务、运营、战略指标
- 定期回顾：季度、半年、年度
- 战略调整：方向调整、资源重新分配

### Step 6: 退出执行
- 退出决策：时机、方式、定价
- 退出执行：IPO、并购、回购
- 投后总结：经验、教训、知识沉淀
