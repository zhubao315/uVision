# 📚 得到APP 网络课程制作规范 (2026-02-11 14:00 UTC)

## 📋 规范核心要点

### 1. 课程结构标准
- **总讲数**: 7讲
- **每讲字数**: 3000–4000字
- **标准结构**:
  - 问题导入 (300字) → 痛点场景 + 悬念
  - 理论讲解 (1200字) → 核心概念 + 原理
  - 案例分析 (1200字) → 3–5个职场案例
  - 行动指南 (300字) → 具体步骤 + 实践任务

### 2. 内容元素要求
| 元素 | 数量/讲 |
|--|--|
| 职场案例 | 3–5个 |
| 图表 | 2–3个 |
| 实践任务 | 1个 |
| 思考题 | 2–3个 |

### 3. 表达风格
- **语言**: 平实易懂，避免学术腔
- **节奏**: 每200字设一个"钩子"
- **互动**: 每讲结尾设问

### 4. 质量审核标准
- **知识准确性**: 无事实错误
- **逻辑严谨性**: 结构清晰
- **实用性**: 提供可执行步骤
- **吸引力**: 有悬念、有故事
